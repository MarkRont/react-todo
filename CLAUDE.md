# React To-Do List App

## Tech Stack
- React 19 with Vite
- Plain CSS (no Tailwind, no CSS frameworks)
- No TypeScript — plain JavaScript (.jsx)

## Project Structure
- `src/App.jsx` — All app logic lives here (components, state, handlers)
- `src/App.css` — All custom styles
- `src/main.jsx` — Entry point (do not modify)
- `index.html` — HTML shell (do not modify)

## Dev Commands
- `npm run dev` — Start dev server (localhost:5173)
- `npm run build` — Production build
- `npm run lint` — Run ESLint

## Guidelines
- Keep it simple: all logic in App.jsx, styles in App.css
- The user is a beginner learning React for the first time
- Explain React concepts (useState, props, components, JSX, event handling) in code comments when introducing them
- Use plain CSS — no utility classes, no CSS-in-JS
- Prefer simple, readable code over clever abstractions
- Avoid creating new files unless explicitly asked
